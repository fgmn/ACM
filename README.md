# ACM 算法模板

## 图论

Dijkstra

Floyd

spfa

spfa_diffConstrainedSys(差分约束系统)

toposort(拓扑排序)

MST(Kruskal求最小生成树)

diameterOfTree(dfs求树的直径)

diameterOfTree_V2(树形DP求树的直径，同时统计直径个数)

Kosaraju(强连通缩点，$O(n ^ 2)$)

## 数据结构

树状数组：
    V1：单点更新、区间查询
    V2：区间更新、单点查询
    V3：区间更新、区间查询

珂朵莉树(set上二分)

堆(通过tag实现动态单点更新，区间查询最大值)

线性结构(单调栈、单调队列)

线段树：
    动态开点
    区间乘法(带懒标记)

并查集(DSU)
